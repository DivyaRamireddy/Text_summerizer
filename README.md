# Simple Text Summarizer

This tool takes a paragraph and returns the most important sentences based on word frequency scoring. It performs a basic form of extractive summarization without using any external NLP libraries.

---

## Features
- Identifies important words based on frequency
- Scores each sentence
- Returns the top N most meaningful sentences
- Works entirely with built-in Python functions
